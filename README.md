## 项目目录分析

> 直接根据 LeetCode 上面的题号来找对应的题目就好了
>
> **code**文件夹里面存放的是对应题目的代码，有Python 和 C++ 两种语言版本
>
> 题解放到 solution 这个文件夹里面
>
> 可能用到的图片放在 pic 这个文件夹里面
